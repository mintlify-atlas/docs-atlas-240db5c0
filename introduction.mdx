# NetworkFlux & MaxFlux Generator

NetworkFlux & MaxFlux Generator is an interactive web application for visualizing **network flow graphs** and computing the **maximum flow** through a network using the Ford-Fulkerson algorithm. Built with React and React Flow, it provides a step-by-step visualization of the algorithm so you can follow the computation at your own pace.

## Key features

<CardGroup cols={2}>
  <Card title="Interactive graph visualization" icon="share-nodes">
    Pan, zoom, and explore your network flow graph rendered with React Flow. Nodes and edges are automatically positioned for clarity.
  </Card>
  <Card title="Ford-Fulkerson max flow" icon="code-branch">
    Computes the maximum flow from source **s** to sink **t** using BFS-based augmenting paths (Edmonds-Karp variant).
  </Card>
  <Card title="Step-by-step navigation" icon="list-ol">
    Walk forward and backward through each augmenting path step. The residual graph updates in real time.
  </Card>
  <Card title="JSON adjacency matrix input" icon="brackets-curly">
    Define any directed weighted graph using a simple JSON 2D array — no GUI graph editor required.
  </Card>
  <Card title="Visual source and sink indicators" icon="circle-nodes">
    The source node **s** displays a server icon; the sink node **t** shows a lamp that lights up when max flow meets the expected value.
  </Card>
  <Card title="Bidirectional edge rendering" icon="arrows-left-right">
    Parallel edges between the same pair of nodes are rendered with a bezier curve offset so they never overlap.
  </Card>
</CardGroup>
